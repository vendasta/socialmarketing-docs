# Does the Social Marketing API support posting?

We currently offer an API for Social Marketing to partners with a **Custom** subscription, however, the Social Marketing API does not currently support posting to social media networks.

To find out more about actions supported by the Social Marketing API, consult the [Social Marketing API documentation](https://www.vendasta.com/documentation/social-marketing).
